# 🐟 Trivia: CEADI Educa

¡Bienvenido a **CEADI Educa**, una experiencia educativa interactiva diseñada para conocer la fauna ictícola de nuestra región! 🌊

Este proyecto combina **diseño de ilustración propia** con desarrollo web para crear un juego de trivia dinámico, educativo y visualmente atractivo.

---

## 🚀 Características principales
- **Dos niveles de dificultad:** Desde peces superficiales hasta los tesoros de las aguas profundas.
- **Sistema de recompensas:** Medallas de oro y plata según tu desempeño.
- **Experiencia Inmersiva:** Sonido ambiental del río y efectos sonoros para aciertos y errores.
- **PWA (Progressive Web App):** ¡Puedes instalarlo en tu celular como una aplicación real!
- **Feedback Visual:** Revelación de ilustraciones originales tras cada respuesta.

---

## 🎨 El Diseño
Como parte de la **Oficina de Producción**, el enfoque principal fue la creación de una interfaz limpia y amigable (UI) que resalte las ilustraciones de los ejemplares:
* Surubí
* Dorado
* Pacú
* ¡Y muchos más!

---

## 🛠️ Tecnologías utilizadas
- **HTML5:** Estructura semántica de las pantallas.
- **CSS3:** Animaciones de confeti, efectos de transición y diseño responsivo.
- **JavaScript (Vanilla):** Lógica de niveles, sistema de puntaje y manejo de estados del juego.
- **Manifest JSON:** Configuración para instalación en dispositivos móviles.

---

## 🕹️ Cómo jugar
1. Ingresa al link del proyecto.
2. Haz clic en **¡EMPEZAR!**.
3. Observa la pregunta y elige la opción correcta.
4. ¡Descubre la ilustración del pez y suma puntos para ganar la medalla de oro!
